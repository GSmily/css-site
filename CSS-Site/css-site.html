CSS Unix Command Cheat Sheet

By Gavin Browning

These are all useful

Unix shell commands!

Commands Descriptions Examples
1. ls Lists files in the current directory
List in a long format		 
ls
ls-alF
2. cd Changes directory to temp
Move back one directory		 
cd temp
cd..
3. cp Copy file into directory
Make backup of temp1		 
cp temp1 web-stuff
cp temp1 temp1.bak
4. rm Delete or remove file
Remove all files		 
rm temp1.bak
rm*.tmp
5. mv Move or rename files 
mv old.txt new.txt
